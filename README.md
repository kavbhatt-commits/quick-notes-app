📝 Quick Notes App

A modern Full Stack Notes Dashboard built using Next.js, React, Node.js, and Express. This project allows users to create, edit, delete, search, filter, pin, and organize notes through a clean, responsive interface.

---

🚀 Features

- Create Notes
- Edit Notes
- Delete Notes
- Search Notes
- Category Filter
- Priority Filter
- Pin / Unpin Notes
- Dashboard Statistics
- Dark / Light Mode
- Export Notes as PDF
- Export Notes as TXT
- Responsive UI
- Glassmorphism Design
- Gradient Dashboard
- Smooth Animations
- Professional SaaS-inspired Interface

---

🛠️ Tech Stack

Frontend

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4

Backend

- Node.js
- Express.js
- REST API
- File System (JSON Storage)

⚙️ Installation

Clone Repository

git clone <repository-url>

Backend

cd backend
npm install
npm start

Backend runs at:

http://localhost:5000

Frontend

cd frontend
npm install
npm run dev

Frontend runs at:

http://localhost:3000

---

📌 API Endpoints

Method| Endpoint| Description
GET| "/api/notes"| Get all notes
POST| "/api/notes"| Create a note
PUT| "/api/notes/:id"| Update a note
DELETE| "/api/notes/:id"| Delete a note
GET| "/api/notes/export/pdf"| Export PDF
GET| "/api/notes/export/txt"| Export TXT
